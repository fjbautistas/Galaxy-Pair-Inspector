🌌 How to classify
Galaxy Pair Inspector
You will see cutout images from the DESI Legacy Survey. Your task is to classify each object into one of 5 categories.

Swipe to learn each one →
✅ Confirmed Pair
Two galaxies physically close in space (small rp, similar redshift). They may show tidal distortion or a connecting bridge. Both galactic centers are still clearly distinguishable as separate objects. Mark as Pair.

🎯 Classify only the marked galaxies. Other galaxies visible in the image are projections at different redshifts — already filtered out using velocity cuts (Δv) and projected distance (rp), so they are not part of this pair.

⚠️ If both markers point to the same object (deblending artifact), it's actually a False Positive.
❌ False Positive
Projection: two galaxies at very different redshifts that overlap on the sky. Not a real pair.

Deblending: the catalog mistakenly split one galaxy into two sources (G1 and G2 point to the same object). Both cases → FP.
🔀 Possible Merger
Two nuclei within a shared envelope — heavily deformed and being pulled toward each other. The individual galactic centers can no longer be clearly separated; they are in the process of merging. Strong tidal tails on opposite sides are the key signature. Mark as PM.
⭐ Confirmed Group
3 or more galaxies physically associated (same redshift, gravitationally bound). The app marks the closest members. Mark as Group.
❌ FP in Groups
The catalog shows a group of N=4, but there is only one real galaxy. All markers (G1–G4) point to the same object — a severe deblending error. There is no group here. Mark as FP.
👥 Possible Pair
Left: two clearly distinct galaxies within a group — they look like a pair visually.

Right: a group of N=3 where G1 and G2 are deblending artifacts of the same galaxy. The catalog says group, but the image shows only 2 real objects. Both cases → PP.
